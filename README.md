# Getting Started

#### What is?

Perform scaffolding, automate build and manage dependencies with adamantium

### install dependences
Yeoman:
```bash
npm install -g yo
```

To install adamantium generator from npm, run:

```bash
npm install -g generator-adamantium
```

Finally, initiate the generator:

```bash
yo adamantium
```
